为什么选择 udplink？

🚀 固定内存

编译时确定，无动态分配，无堆内存需求，MCU 友好

纯 C++11 实现，无外部库依赖，易于移植

🔒 安全可靠

SipHash 鉴权、双密钥在线换钥、防重放攻击

📦 商用扩展

可插拔网络适配层、可配置窗口/重传策略

快速开始

# 克隆项目
git clone https://github.com/KairuiKin/udplink.git
cd udplink

# 编译
cmake -S . -B build
cmake --build build --config Release

# 测试
./build/rudp_self_test

代码示例

#include "rudp/rudp.hpp"

rudp::Config config = rudp::ConfigForProfile(rudp::kProfileLowPower);
config.enable_auth = true;
config.auth_key0 = 0x0123456789ABCDEF;
config.auth_key1 = 0xFEDCBA9876543210;

rudp::Hooks hooks;
hooks.send = [](const uint8_t* d, uint32_t len, void*) { 
    return udp_send(d, len); 
};
hooks.get_tick_ms = []() { return now_ms(); };

rudp::Endpoint endpoint;
endpoint.Init(config, hooks);
endpoint.StartConnect("192.168.1.100", 8888);
